# 記事投稿Webアプリ

## 機能
- 記事投稿
- 記事更新
- 画像投稿
- 記事一覧表示
- コメント投稿


## 使い方

#### 起動
```
docker-compose up
```
最初は起動に失敗するのでCtrl+Cで停止して、もう一度起動してください。

接続先：[localhost:8001](http://localhost:8001/)  
#### 停止
```
docker-compose down
```
